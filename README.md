# 应用概述

Flutter-PicGo: 一个用于快速上传图片并获取图片URL链接的**手机版**工具

Flutter-PicGo 本体支持如下图床：

- GitHub [v1.0]
- SM.MS [v1.1]
- Gitee [v1.2]

> 开发进度可以查看 Projects，会同步更新开发进度
